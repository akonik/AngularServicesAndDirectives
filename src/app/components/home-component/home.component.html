Task description :

Using the Angular 2+ / JS framework, create SPA that should contain information based on your resume.

SPA should consist of several sections.

In addition to the text description, the data should be presented in the form of various diagrams (you can libraries, for example, charts.js)

When building, you must use services, directives.

Service (DataService) - provides all the data for resumes and charts.

Directive (HoverDirective) - changes the background color of the element when the mouse enters the component, and returns it to the default color when the mouse leaves.
